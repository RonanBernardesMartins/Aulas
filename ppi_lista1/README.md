# PPI1 - Lista 1: Fundamentos da Web e HTML

## Descrição do Projeto
Este projeto consiste em uma série de exercícios práticos de HTML e CSS, com o objetivo de fixar os fundamentos da Web, estrutura de documentos HTML, formulários, tabelas, links, imagens e estilização via CSS externo.  

O projeto foi desenvolvido para fins acadêmicos da disciplina **Programação para Internet 1 (PPI1)**.

## Estrutura de Pastas e Arquivos

ppi1_lista1/
├── 1.html # Estrutura básica HTML e história da Internet/Web
├── 2_http.html # Explicação sobre o protocolo HTTP
├── 3_tabela_uf.html # Tabela de estados brasileiros com links
├── 4_curriculo.html # Currículo com seções, tabela de horários e navegação interna
├── 4_contato.html # Página de contato com formulário funcional
├── 5_links.html # Página de links (opcional)
├── 6_github.html # Página explicando publicação no GitHub Pages
├── style.css # Arquivo de estilos CSS externo
└── README.md # Este arquivo explicativo

## Conteúdo das Atividades

### Atividade 1 – Estrutura Básica HTML
- Criação do arquivo `1.html` com:
  - Estrutura mínima HTML (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
  - Texto explicativo sobre a diferença entre Internet e Web (máx. 5 linhas)
  - Comentário indicando autor e data
  - Seção `<div>` contendo história da Internet e Web com:
    - Principais marcos históricos
    - Uso de imagens `<img>`
    - Lista de acontecimentos `<ul>`, `<ol>`
    - Marcação de texto `<em>`, `<mark>`, `<strong>`, `<sub>`, `<sup>`
    - Tabelas `<table>` com `<thead>`, `<tbody>`, `<tfoot>` e `<caption>`

### Atividade 2 – Protocolo HTTP
- Criação do arquivo `2_http.html` explicando:
  - O que é HTTP
  - Como funciona na Web
  - Exemplos de requisições e respostas (request/response)
  - Campos de um request HTTP detalhados

### Atividade 3 – Tabela de Estados
- Criação do arquivo `3_tabela_uf.html` com:
  - Tabela contendo: índice, região, nome, estado, sigla, capital
  - Agrupamento de linhas por região (`rowspan`)
  - Links para Wikipedia de cada estado

### Atividade 4 – Currículo HTML
- Arquivo `4_curriculo.html` com:
  - Nome completo e foto
  - Contato com links funcionais (`mailto:` e `tel:`)
  - Breve descrição profissional
  - Lista de habilidades técnicas (`<ul>` ou `<ol>`)
  - Formação acadêmica (`<section>`, `<article>` ou `<div>`)
  - Experiência profissional com marcações de texto (`<strong>`, `<em>`, `<mark>`)
  - Tabela de horários fictícia
  - Menu de navegação interno usando âncoras
  - Uso de elementos `<div>`, `<span>`, `<p>`, `<b>`, `<em>`, `<strong>`, `<mark>`, `<sub>`, `<details>`, `<summary>`
  - Links externos (ex.: LinkedIn)

### Atividade 5 – Página de Contato
- Arquivo `4_contato.html` com:
  - Formulário com campos: Nome, E-mail, Assunto, Estado, Mensagem
  - Labels e ids para cada input
  - Validação HTML (`required`, `type=email`)
  - Placeholders
  - Botão de envio estilizado
  - Link de navegação para `4_curriculo.html`

### Atividade 6 – Aplicação de CSS Externo
- Arquivo `style.css` utilizado por todas as páginas
- Estilos aplicados:
  - `background-color` para fundo da página
  - `color` para texto principal
  - `font-family: Arial, sans-serif;`
  - Estilização de tabelas (`border`, `padding`)
  - Inputs arredondados (`border-radius`), margens
  - Botões com cores e efeito hover (`background-color`, `:hover`)
  - Links do menu com efeito hover

### Atividade 7 – Publicação no GitHub Pages
- Repositório hospedado no GitHub
- GitHub Pages ativado para visualização online
- Link do projeto publicado compartilhado
